## 依赖边说明

| From | To | Reason |
|------|-----|--------|
| 01-Hypergraph Construction | 02-Graph Normalization | 归一化是对邻接矩阵的数学变换，必须先有矩阵 |
| 02-Graph Normalization | 03-HGNN Layer | HGNN 的消息传递依赖归一化后的邻接矩阵 |
| 01-Hypergraph Construction | 09-Graph-Aware Evaluation | SSIM 需要原始邻接矩阵定义"局部" |
| 03-HGNN Layer | 05-Single-Slice Model | 单 slice 模型使用 HGNN 作为核心编码器 |
| 04-MLP Encoder | 05-Single-Slice Model | 模型需要先通过 MLP 将 HE 嵌入投影到隐藏维度 |
| 05-Single-Slice Model | 06-Pseudo-Spot Aggregation | 聚合是对模型输出的后处理，需要先有预测 |
| 05-Single-Slice Model | 07-Regression Translator | 翻译器的输入是单 slice 模型的预测结果 |
| 06-Pseudo-Spot Aggregation | 08-Six-Loss Trainer | 训练循环需要 spot-level loss 作为监督信号 |
| 07-Regression Translator | 08-Six-Loss Trainer | 6-loss 中的 loss3-6 依赖翻译器 |
| 03-HGNN Layer | 10-Graph Transformer | GT 是 HGNN 的注意力升级，先懂 HGNN 才能理解改进 |
| 02-Graph Normalization | 10-Graph Transformer | GT 的稀疏邻接矩阵需要先归一化 |
